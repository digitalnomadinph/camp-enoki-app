CAMP ENOKI - INSTALLABLE APP (PWA)
==================================

These files wrap your booking portal so it can be INSTALLED like an app on
a phone (custom icon, fullscreen, no address bar).

Files: index.html, manifest.webmanifest, service-worker.js, icon-192.png,
icon-512.png   (make-icons.js is just the icon generator - not needed online)

BEFORE PUBLISHING
-----------------
In the Apps Script editor: Deploy > Manage deployments > (edit) >
"Who has access" must be ANYONE, so guests can open it without logging in.


=== OPTION A: NETLIFY DROP (easiest, ~2 minutes, free, no setup) ===
1. On a computer, open:  https://app.netlify.com/drop
2. Drag the whole "camp-enoki-pwa" FOLDER onto that page.
3. It gives you an https link like  random-name.netlify.app
4. (Free sign-up keeps the link permanent; you can rename it in Site settings.)
5. Open that link on your phone in Chrome -> menu -> "Install app" /
   "Add to Home screen". Done - it installs with the Camp Enoki icon.


=== OPTION B: GITHUB PAGES (free) ===
1. github.com -> New repository (Public) -> e.g. "camp-enoki-app".
2. "uploading an existing file" -> drag these 5 files in -> Commit.
3. Repo Settings -> Pages -> Source: Deploy from a branch -> main -> /(root) -> Save.
4. After ~1 min it gives:  https://YOURNAME.github.io/camp-enoki-app/
5. Open on phone -> Install app.


NOTES
-----
- The icon is a placeholder (green tent). Send your logo to swap it in.
- If you ever create a brand-new Apps Script deployment, update the iframe
  "src" URL in index.html to the new /exec link, then re-publish.
